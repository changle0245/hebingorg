如何在线合并多段 MP3 音频文件

  1. 上传音频 — 点击或拖拽多段 MP3 / WAV / M4A 文件到上传区域。
  2. 调整顺序 — 拖动文件列表，决定音频段的先后顺序。
  3. 点击合并 — 浏览器使用 Web Audio API 本地按顺序拼接音频。
  4. 下载文件 — 合并后的音频自动下载，可在任何播放器中播放。

使用小贴士

  • 不同采样率的音频会自动统一到最高采样率输出，避免音质损失。
  • 合并前建议各段音频保持相同声道数（单声道/立体声），否则可能出现声道错位。
  • 如果只是想裁剪或淡入淡出，建议使用专业音频编辑器；本工具专注纯拼接。
  • 合并大量长音频（如 2 小时以上）时浏览器内存占用较高，建议分批合并。

关于音频合并的常见问题

合并后的音频是什么格式？
默认输出 MP3（192kbps CBR，lamejs 浏览器本地编码），体积小、兼容所有播放器。也可一键切换为 WAV 16bit 无损格式。
支持哪些音频格式？
支持 MP3、WAV、M4A、AAC、OGG、FLAC 等浏览器 Web Audio API 可解码的所有音频格式。
合并会损失音质吗？
如果所有源音频格式一致、采样率一致，合并是无损拼接。如果格式不同会统一重编码，在高码率下听感上无明显差异。
可以在合并处加淡入淡出吗？
目前不支持。本工具专注纯拼接，不做剪辑。如需高级效果建议使用 Audacity 等专业软件。
文件会上传服务器吗？
不会。所有音频在浏览器的 Web Audio API 中本地处理，音频数据绝不上传。
能合并多长的音频？
理论上没有硬性上限，取决于设备内存。一般建议单次合并总时长不超过 3 小时，以获得流畅体验。
