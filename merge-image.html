如何将多张图片合并为 PDF 或长图

  1. 上传图片 — 点击或拖拽多张图片到上传区域，支持 JPG、PNG、WebP。
  2. 选择输出格式 — 选择合并为 PDF 文档，或合并为垂直拼接的长图（PNG/JPG）。
  3. 调整顺序 — 拖动列表中的图片决定排列先后。
  4. 下载结果 — 浏览器本地完成合并，文件自动下载。

使用小贴士

  • 合并后图片会保留原始分辨率，但合并为长图时若图片过大可能被浏览器内存限制。
  • 需要缩小单张图片体积时，可先用 图片压缩 工具处理再合并。
  • 长图输出默认垂直拼接，宽度以第一张图片为基准，其他图片自动等比缩放。
  • PDF 输出每张图片对应一页，适合打印或作为附件发送。

关于图片合并的常见问题

图片合并后是什么格式？
你可以选择合并为 PDF 文档（每张图一页），或合并为垂直拼接的长图（PNG 或 JPG）。
支持哪些图片格式？
支持 JPG、JPEG、PNG、WebP、GIF（仅首帧）等浏览器原生支持的格式。
合并后画质会下降吗？
不会。原始图片像素完整保留，合并仅做排版，不做重新压缩。
长图的宽度如何决定？
以第一张图片的宽度为基准，其他图片自动等比缩放对齐，从上到下顺序排列。
文件会上传到服务器吗？
不会。整个合并过程在浏览器的 Canvas API 中本地完成，图片数据不离开设备。
最多可以合并多少张图片？
没有强制上限。输出长图时建议不超过 30 张（避免超出浏览器 Canvas 最大尺寸），输出 PDF 可支持更多。
